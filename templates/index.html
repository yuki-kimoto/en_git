Gitコマンド

  • git init - Gitリポジトリの初期化
  • git log - コミットログの内容を見る
  • git status - コミット準備されているかを確認する
  • git add - ファイルをコミット準備状態にする
  • git commit - コミットする
  • git branch - ブランチの一覧表示・ブランチの作成
  • git checkout - ブランチの変更
