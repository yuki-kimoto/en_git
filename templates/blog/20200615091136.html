    What is Github Github is a free web service that allows you to manage your Git repolitri. You can also privately manage files that you do not want to expose to third parties. Even with the free plan, you can create private repositories and add unlimited collaborative developers. If you sign up for an enterprise plan for $ 21 / month (January 8, 2019 information), IP address restrictions possible am. It is possible to prevent mistakes such as accidentally making the repository public and leaking confidential information by operating on the Web. The function of Github is mainly source code management, but since there is a bulletin board function called Issue for chasing bugs, it is possible to share information for bug fixes and product improvements. There is also a Wiki feature for each project, and it is possible to share documents using the notation of markdown. If you just use Wiki and Issue, you don't need to know Git, so even if you don't have Git knowledge, it's worth a try.

    Register an account on Github First, account registration to Github. When you register for an account, you will receive an email. Click the Verify email address link to complete the registration. When the sign-in screen appears, sign in with your registered user name and password. User authentication is complete.
