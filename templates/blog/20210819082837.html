What is a merge?

Merging is the incorporation of changes made to the source code. For example, suppose you have a file called "a.txt" in my current main branch. Contents of "a.txt" in my current main branch

Merge algorithm

There are multiple algorithms applied to merging.

Fast forward merge

The simplest merge algorithm. If a merge is possible by simply proceeding with the commit, this will be selected.

Non-fast forward merge

Merges the differences from the positions of the common ancestors of the two branches. Marges may conflict.
